# Hephzibah K. - Full Stack & ML Engineer Portfolio 🚀

> A beautiful, responsive, and highly interactive 3D portfolio showcasing my skills, experience, and flagship AI/Full Stack projects. Built with React, Three.js, and Framer Motion.

## 🌟 Key Features
- **Quantum-Themed 3D Scene:** Immersive interactive background using `@react-three/fiber` and `@react-three/drei`. Includes camera parallax, a glowing core planet, and floating asteroid belt.
- **Fluid Animations:** Smooth micro-interactions and scroll-based section reveals powered by React Framer Motion.
- **Glassmorphism UI:** Modern, slick, and accessible translucent UI components over a deep space background.
- **Fully Responsive:** Device-optimized structure with a seamless mobile hamburger menu and smart layout collapsing.
- **Serverless Contact Form:** Secure email dispatch utilizing Resend API hosted securely on a Vercel Serverless Function (`api/contact.js`).

## 🛠️ Tech Stack
- **Frontend Framework:** React (Vite)
- **3D Graphics:** Three.js, React Three Fiber
- **Animations:** Framer Motion
- **Styling:** Vanilla CSS3, Grid/Flexbox, Dynamic Variables
- **Icons:** Lucide React
- **Backend/Deployment:** Vercel (Hosting & Serverless Functions)
- **Email Delivery:** Resend API

## 🚀 Quick Start (Local Development)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/hepsi33/[your-repo-name].git
   cd [your-repo-name]
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the local development server:**
   ```bash
   npm run dev
   ```

*(Note: The `/api/contact` email form utilizes Vercel Serverless Functions and will only dispatch emails when deployed to Vercel, or tested locally using the Vercel CLI).*
